## Contact List

Home/Landing Page

- You can search a contact by name, their details will render below.
- Click the "All Contacts" link to take you to the page with all your saved contacts.

All Contacts Page

- Click Home to return to the Home/Landing page
- All your contacts will display in a table on this page
- Clicking "Delete" will remove the contact from your list.
- Click the "Add Contact" button to take you to the add contact form.
- If you click on a contact's name, it will take you to the "Contact info" page.
- EDIT button does not have functionality at the time of this pull request.

Add New Contact Page

- enter all inputs and click the "Add New Contact" button to submit.
- Click the back link to return to the All Contacts page.

Contact Info Page

- Will show the contact info specific to the name that you clicked previously.
- Click the back link to return to the All Contacts page.
